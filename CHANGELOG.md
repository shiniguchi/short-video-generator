### Added
- Rate limiting and API key validation
- Google image and video providers using google-genai SDK
- UGC pipeline with viral Hook-Problem-Proof-CTA framework
- Avatar generation via HeyGen integration
- Video generation via FAL (Kling, MiniMax) and SVD providers
- Voiceover generation via ElevenLabs, OpenAI TTS, and Fish Audio
- Video compositor with text overlay, audio mixing, and thumbnails
- Trend collection and analysis from TikTok and YouTube
- FastAPI REST API with Celery task queue
- Docker Compose setup with PostgreSQL and Redis
- SQLite fallback for local development
