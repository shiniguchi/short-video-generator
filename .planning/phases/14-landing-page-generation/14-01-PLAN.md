  <action>
Create app/services/landing_page/research.py implementing the research stage of the LP generation pipeline. This module scrapes top-performing LPs in the same industry/region and extracts design patterns.

Key functions:

1. `async def research_competitor_lps(industry: str, region: str = "US", count: int = 10) -> LPResearchResult`:
   - Main entry point. Uses Playwright to search for top LPs in the industry/region.
   - Search strategy: Google search for "best {industry} landing pages {region} 2026" and extract result URLs.
   - For each URL (up to `count`), navigate with `wait_until="networkidle"`, extract HTML, parse with BeautifulSoup.
   - Extract pattern data using helper functions below.
   - Rate limit: 3-second delay between page loads to avoid blocking.
   - Set User-Agent header: "ViralForge-LPResearch/1.0".
   - Handle errors gracefully (skip individual URLs that fail, log warnings, continue with remaining).
   - Aggregate patterns into LPResearchResult with common_sections, dominant_cta_style, video_placement_trend, color_trends.
   - Use `from typing import List, Optional, Dict` for Python 3.9 compat.

2. Helper extraction functions (private):
   - `_extract_hero_headline(soup: BeautifulSoup) -> str` — find first h1, return text.
   - `_extract_cta_buttons(soup: BeautifulSoup) -> List[str]` — find a/button elements with CTA-like classes (cta, btn, button, signup, subscribe, join).
   - `_extract_section_order(soup: BeautifulSoup) -> List[str]` — identify major sections by tag (header/section/footer) or class names containing hero/features/benefits/pricing/testimonial/faq/cta/footer.
   - `_extract_video_placement(soup: BeautifulSoup) -> Optional[str]` — detect video tag presence, determine if in hero section ("hero"), elsewhere ("middle"), or absent (None).
   - `_extract_color_scheme(soup: BeautifulSoup) -> Optional[dict]` — parse inline styles and CSS for background-color, color properties. Return dict of {primary, secondary, accent} hex values if found.
   - `_aggregate_patterns(patterns: List[LPResearchPattern]) -> dict` — compute most common sections, CTA styles, video placement trend across all patterns.

3. `def get_mock_research(industry: str, region: str) -> LPResearchResult`:
   - Returns realistic mock data for development without Playwright/internet. Follow the project's mock-first pattern.
   - Include 3 mock patterns with realistic headline/CTA/section data.
   - Used when USE_MOCK_DATA=True or Playwright not available.

4. `async def research_lps(industry: Optional[str], region: Optional[str], use_mock: bool = False) -> LPResearchResult`:
   - Top-level dispatcher: if use_mock or industry is None, return mock data. Otherwise call research_competitor_lps.
   - This is what the pipeline calls.

Import LPResearchPattern and LPResearchResult from app.schemas. Use logging throughout. Wrap Playwright operations in try/except to handle browser launch failures gracefully (fall back to mock).
