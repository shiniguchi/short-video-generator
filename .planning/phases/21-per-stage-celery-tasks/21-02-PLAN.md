<task type="auto">
  <name>Task 1: Create UGC router with submit, advance, and status endpoints</name>
  <files>
    app/ugc_router.py
  </files>
  <action>
Create `app/ugc_router.py` with three FastAPI endpoints on an `APIRouter(prefix="/ugc", tags=["ugc"])`.

**Endpoint 1: POST /ugc/jobs — Submit new UGC job**
- Accept `Form` fields: `product_name: str`, `description: str`, `use_mock: bool = True`, `product_url: Optional[str] = None`, `target_duration: int = 30`, `style_preference: Optional[str] = None`.
- Accept `images: List[UploadFile] = File(default=[])` for product images.
- Save uploaded images to `output/ugc_uploads/{job_id}/` directory. Store paths in `product_image_paths` JSON column.
- Create UGCJob row with all input fields and `status="pending"`.
- Transition `pending -> running` via `UGCJobStateMachine.send("start")`.
- Commit the job to DB.
- Enqueue `ugc_stage_1_analyze.delay(job.id)`.
- Return `{"job_id": job.id, "status": job.status}` as JSON.
- Use `session: AsyncSession = Depends(get_session)` for DB access (FastAPI context, not task context).

**Endpoint 2: POST /ugc/jobs/{job_id}/advance — Advance past review stage**
- Load UGCJob by ID. Return 404 if not found.
- Check if `job.status` is in the advance map. Return 400 if not a review state.
- Advance map — maps current review status to (approve_event, next_task_function_name):
  ```python
  _STAGE_ADVANCE_MAP = {
      "stage_analysis_review": ("approve_analysis", "ugc_stage_2_script"),
      "stage_script_review":   ("approve_script",   "ugc_stage_3_aroll"),
      "stage_aroll_review":    ("approve_aroll",     "ugc_stage_4_broll"),
      "stage_broll_review":    ("approve_broll",     "ugc_stage_5_compose"),
      "stage_composition_review": ("approve_final",  None),
  }
  ```
- Send the approve event via state machine (transitions to `running` or `approved`).
- If `approve_final`, set `job.approved_at = datetime.now(timezone.utc)`.
- Commit.
- If `next_task_name` is not None, lazy-import `app.ugc_tasks`, get the task function via `getattr`, call `.delay(job.id)`.
- Return `{"job_id": job.id, "status": job.status, "next_stage": next_task_name}`.

**Endpoint 3: GET /ugc/jobs/{job_id} — Get job status and outputs**
- Load UGCJob by ID. Return 404 if not found.
- Return a JSON dict with all relevant fields: `id`, `product_name`, `status`, `use_mock`, `created_at`, `error_message`, plus all stage output columns (analysis_*, hero_image_path, master_script, aroll_scenes, broll_shots, aroll_paths, broll_paths, final_video_path, cost_usd, approved_at).
- Use `jsonable_encoder` or manual dict construction to serialize datetime fields.

**Error handling:**
- Import `TransitionNotAllowed` from `statemachine.exceptions` and catch it in the advance endpoint. Return 400 with the error message.
- Catch `ValueError` from missing job and return 404.
