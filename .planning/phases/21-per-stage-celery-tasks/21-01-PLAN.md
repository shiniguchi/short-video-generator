  <action>
Create `app/ugc_tasks.py` with five `@celery_app.task` functions. Each task follows the same shape: load UGCJob from DB, validate state via state machine, run one service function, write output to UGCJob columns, transition to review status, commit.

**Design decision:** Stages 1+2 (analysis + hero image) are combined into one task `ugc_stage_1_analyze` because the state machine has no `stage_hero_image_review` state — both complete before `stage_analysis_review`. This avoids adding new states or a migration.

**All five tasks:**

1. `ugc_stage_1_analyze(self, job_id: int)` — runs `analyze_product()` and `generate_hero_image()`, writes all `analysis_*` columns + `hero_image_path`, sends `complete_analysis` -> `stage_analysis_review`.

2. `ugc_stage_2_script(self, job_id: int)` — runs `generate_ugc_script()`, writes `master_script` (as dict via `.model_dump()`), `aroll_scenes` (list of dicts), `broll_shots` (list of dicts), sends `complete_script` -> `stage_script_review`.

3. `ugc_stage_3_aroll(self, job_id: int)` — runs `generate_aroll_assets()`, writes `aroll_paths`, sends `complete_aroll` -> `stage_aroll_review`.

4. `ugc_stage_4_broll(self, job_id: int)` — runs `generate_broll_assets()`, writes `broll_paths`, sends `complete_broll` -> `stage_broll_review`.

5. `ugc_stage_5_compose(self, job_id: int)` — runs `compose_ugc_ad()`, writes `final_video_path` and `cost_usd`, sends `complete_composition` -> `stage_composition_review`.

**Task decorator pattern (copy from existing tasks.py):**
```python
@celery_app.task(
    bind=True,
    name='app.ugc_tasks.ugc_stage_1_analyze',
    max_retries=1,
    time_limit=600,
)
```
Use `time_limit=600` (10 min) for stages 1-4, `time_limit=1200` (20 min) for stage 5 (composition is slower).

**DB access pattern (from compose_video_task):**
```python
import asyncio

async def _run():
    from app.database import get_task_session_factory
    from app.models import UGCJob
    from sqlalchemy import select

    session_factory = get_task_session_factory()
    async with session_factory() as session:
        result = await session.execute(select(UGCJob).where(UGCJob.id == job_id))
        job = result.scalars().first()
        if not job:
            raise ValueError(f"UGCJob {job_id} not found")
        # ... run service, write columns, transition state, commit

asyncio.run(_run())
```

**State machine usage:**
```python
from app.state_machines.ugc_job import UGCJobStateMachine
sm = UGCJobStateMachine(model=job, state_field="status", start_value=job.status)
sm.send("complete_analysis")
# If sm.send() doesn't update job.status via model binding, fallback:
# job.status = sm.current_state.id
await session.commit()
```

**Error handling:** Each task wraps the async _run in try/except. On failure, transition to `failed` via state machine and write `job.error_message`. Pattern:
```python
try:
    asyncio.run(_run())
except Exception as exc:
    asyncio.run(_fail_job(job_id, str(exc)))
    raise
```

Write a separate `_fail_job(job_id, error_msg)` async helper at module top that loads the job, sends `fail` event, writes error_message, and commits.

**Service call specifics per stage:**

Stage 1 (analyze + hero):

- `generate_hero_image(product_image_path=(job.product_image_paths or [""])[0], ugc_style=analysis.ugc_style, emotional_tone=analysis.emotional_tone, visual_keywords=analysis.visual_keywords, use_mock=job.use_mock)`
- Write: `job.analysis_category`, `job.analysis_ugc_style`, `job.analysis_emotional_tone`, `job.analysis_key_features`, `job.analysis_visual_keywords`, `job.analysis_target_audience`, `job.hero_image_path`

Stage 2 (script):
- Reconstruct `ProductAnalysis` from job columns: `ProductAnalysis(category=job.analysis_category, key_features=job.analysis_key_features, target_audience=job.analysis_target_audience, ugc_style=job.analysis_ugc_style, emotional_tone=job.analysis_emotional_tone, visual_keywords=job.analysis_visual_keywords)`
- `generate_ugc_script(product_name=job.product_name, description=job.description, analysis=analysis, target_duration=job.target_duration, use_mock=job.use_mock)`
- Write: `job.master_script = breakdown.master_script.model_dump()`, `job.aroll_scenes = [s.model_dump() for s in breakdown.aroll_scenes]`, `job.broll_shots = [s.model_dump() for s in breakdown.broll_shots]`

Stage 3 (aroll):
