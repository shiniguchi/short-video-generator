<task type="auto">
  <name>Task 2: Create Worker handler with /track and /analytics routes</name>
  <files>workers/lp-analytics/src/index.js</files>
  <action>
Create `src/index.js` with a single default export `{ fetch(request, env, ctx) }`.

**Route handling (no framework, URL + method matching):**

1. **OPTIONS (any path)** — Return 204 with CORS headers (Access-Control-Allow-Origin: *, Allow-Methods: POST GET OPTIONS, Allow-Headers: Content-Type Authorization).

2. **POST /track** — `handleTrack(request, env, ctx)`:
   - Parse body via `request.text()` then `JSON.parse()` (NOT `request.json()` — sendBeacon sends text/plain content-type).
   - Extract `lp_id`, `event`, `referrer` from body.
   - If missing `lp_id` or `event`, return 400 with CORS.
   - Use `ctx.waitUntil()` for non-blocking D1 write:
     - If `event === "pageview"`: INSERT INTO pageviews (lp_id, referrer, tracked_at) VALUES (?, ?, Date.now())
     - If `event === "form_submit"`: INSERT INTO form_submissions (lp_id, tracked_at) VALUES (?, Date.now())
   - Return 204 with CORS immediately (before write completes).

3. **GET /analytics/:lp_id** — `handleAnalytics(request, env, lp_id)`:
   - Check `Authorization` header === `Bearer ${env.API_KEY}`. If mismatch, return 401.
   - Use `env.DB.batch()` for parallel queries:
     - COUNT(*) from pageviews WHERE lp_id = ?
     - COUNT(*) from form_submissions WHERE lp_id = ?
   - Separate query for top 10 referrers: SELECT referrer, COUNT(*) as count FROM pageviews WHERE lp_id = ? GROUP BY referrer ORDER BY count DESC LIMIT 10
   - Return JSON: `{ lp_id, pageviews, form_submissions, top_referrers }` with 200 + CORS.

4. **All other paths** — Return 404.

**CORS helper function** `corsResponse(body, status)`:
- Sets Access-Control-Allow-Origin: *, Allow-Methods, Allow-Headers, Content-Type.
- body=null → empty Response.
  </action>
