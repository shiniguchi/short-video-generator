Add two endpoints to the existing ugc_router.py:

**1. GET /ugc/jobs — List all jobs**

Add `list_ugc_jobs` endpoint before the existing `submit_ugc_job`. Query `select(UGCJob).order_by(UGCJob.created_at.desc())`. Return list of dicts with: id, product_name, status, use_mock, created_at, error_message.

**2. GET /ugc/jobs/{job_id}/events — SSE progress stream**

Add `ugc_job_events` endpoint at the bottom of the file. Takes `job_id: int` and `request: Request` params. No `Depends(get_session)` — the generator must NOT hold a session for the full stream duration.

Inside the endpoint, define `async def event_stream()` generator:
- Loop up to 600 iterations (10 min max)
- At TOP of each iteration: `if await request.is_disconnected(): break` (it's a coroutine — MUST await)
- Open a fresh session per iteration via `async with async_session_factory() as s:` (import from `app.database`)
- Query `select(UGCJob).where(UGCJob.id == job_id)` inside the session context manager
- If job not found: yield `data: {"status": "not_found"}\n\n` and break
- Yield `data: {"status": job.status, "error": job.error_message}\n\n`
- If `job.status` is in terminal states, break
- `await asyncio.sleep(1)` at end of loop

Terminal states: `{"stage_analysis_review", "stage_script_review", "stage_aroll_review", "stage_broll_review", "stage_composition_review", "approved", "failed"}`

Return `StreamingResponse(event_stream(), media_type="text/event-stream", headers={"Cache-Control": "no-cache", "X-Accel-Buffering": "no"})`.

Add imports at top: `import json`, `import asyncio`, `from fastapi import Request`, `from fastapi.responses import StreamingResponse`, `from app.database import async_session_factory` (add to existing database import line).
