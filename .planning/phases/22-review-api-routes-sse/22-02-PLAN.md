**1. Define _STAGE_REGEN_MAP (after _STAGE_ADVANCE_MAP)**

```python
_STAGE_REGEN_MAP = {
    "stage_analysis_review": "ugc_stage_1_analyze",
    "stage_script_review":   "ugc_stage_2_script",
    "stage_aroll_review":    "ugc_stage_3_aroll",
    "stage_broll_review":    "ugc_stage_4_broll",
    # stage_composition_review excluded — no path back to running
}
```

Derive `_REVIEW_STATES = set(_STAGE_ADVANCE_MAP.keys())` for use in edit endpoint gate.
