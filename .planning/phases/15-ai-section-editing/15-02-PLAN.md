**Step A: Create `scripts/edit_lp_section.py`**

Follow the same pattern as `scripts/generate_landing_page.py` (sys.path setup, DATABASE_URL override, logging, argparse).

CLI arguments:
- `--lp` (required unless --list): Path to landing-page.html
- `--section`: Section to edit. Choices: hero, benefits, features, how_it_works, cta_repeat, faq, waitlist, footer. NOT required (only needed for edit, not for --list).
- `--prompt`: Edit instruction (e.g., "make headline shorter", "add urgency"). Required when editing.
- `--product`: Product idea for AI context. Optional — falls back to sidecar file.
- `--list`: List all sections in the LP and exit. Only requires `--lp`.
- `--mock`: Use mock AI (no API key needed).
- `--no-open`: Skip opening browser after edit.

Behavior:

1. **--list mode:** If `--list` flag provided:
   - Read HTML from `--lp` path
   - Call `list_sections(html)` from section_editor
   - Print each section with whether it's editable (compare against `get_editable_sections()`)
   - Exit

2. **Edit mode** (default):
   - Require `--section` and `--prompt`
   - Resolve product_idea: use `--product` if provided, else try reading `landing-page.json` sidecar next to the HTML file (same directory, key: "product_idea"), else error with helpful message
   - Call `edit_section(html_path, section_name, user_prompt, product_idea, use_mock)`
   - Print result (success/failure, warnings)
   - Open browser unless `--no-open`

3. **Sidecar read logic:**
   ```python
   def _read_sidecar(html_path: str) -> dict:
       """Read landing-page.json sidecar if it exists."""
       sidecar_path = Path(html_path).parent / "landing-page.json"
       if sidecar_path.exists():
           import json
           return json.loads(sidecar_path.read_text(encoding='utf-8'))
       return {}
   ```

4. **Sidecar write logic** — update `generator.py` to write sidecar:
   Actually, DON'T modify generator.py in this plan. Instead, have the CLI write the sidecar on first edit if `--product` is provided and sidecar doesn't exist yet. This way:
   - First edit: `--product "Smart Water Bottle"` → edits section AND writes sidecar
   - Subsequent edits: `--product` not needed, reads from sidecar
   ```python
   def _write_sidecar(html_path: str, product_idea: str):
       """Write landing-page.json sidecar for future edits."""
       sidecar_path = Path(html_path).parent / "landing-page.json"
       if not sidecar_path.exists():
           import json
           sidecar_path.write_text(json.dumps({
               "product_idea": product_idea
           }, indent=2), encoding='utf-8')
   ```
