  <action>
Add three new route handlers to router.py:

1. **GET /ui/dashboard** — Analytics dashboard with per-LP stats
   - Accept optional `start` and `end` query params (YYYY-MM-DD strings)
   - Parse dates: `date.fromisoformat(start)` -> `datetime(y,m,d, tzinfo=timezone.utc)`. For end date, add `timedelta(days=1)` to make it inclusive.
   - Query: `select(WaitlistEntry.lp_source, func.count(WaitlistEntry.id).label("count")).group_by(WaitlistEntry.lp_source)` with optional date WHERE clauses on `signed_up_at`. Build `signups_by_lp = {row.lp_source: row.count}` dict.
   - Query: `select(LandingPage).order_by(LandingPage.created_at.desc())` to get all LPs.
   - Fetch analytics concurrently: `from app.services.analytics.client import CloudflareAnalyticsClient` (lazy import inside route body). Create client, then `analytics_list = await asyncio.gather(*[client.get_lp_analytics(lp.run_id) for lp in lps])`. Build `analytics = {lp.run_id: data for lp, data in zip(lps, analytics_list)}`.
   - Return TemplateResponse for "dashboard.html" with context: `lps`, `signups_by_lp`, `analytics`, `start` (original string or ""), `end` (original string or "").

2. **GET /ui/waitlist** — Full waitlist signups table
   - Accept optional `start` and `end` query params (same parsing as dashboard).
   - Query: `select(WaitlistEntry).order_by(WaitlistEntry.signed_up_at.desc())` with optional date WHERE clauses.
   - Return TemplateResponse for "waitlist.html" with context: `entries`, `start`, `end`.

3. **GET /ui/waitlist/export.csv** — CSV download
   - Accept optional `start` and `end` query params (same parsing).
   - Query WaitlistEntry with date filters.
   - Use `io.StringIO` + `csv.writer` in a generator function. First yield: header row "email,signed_up_at,lp_source". Then for each entry: yield row with `e.email`, `e.signed_up_at.isoformat() if e.signed_up_at else ""`, `e.lp_source or ""`.
   - Return `StreamingResponse(generate(), media_type="text/csv", headers={"Content-Disposition": 'attachment; filename="waitlist.csv"'})`.

Add required imports at top: `from datetime import datetime, timezone, date, timedelta`, `from typing import Optional`, `import csv`, `import io`, `from sqlalchemy import func`.
Add `from app.models import LandingPage, WaitlistEntry` (update existing LandingPage import).

Extract a helper `_parse_date_range(start_str, end_str)` that returns `(start_dt, end_dt)` tuple, handling None values.
