## Tech Debt

### Phase 25: LP Integration

- **SSE raw JSON in progress div**: `sse-swap="message"` injects raw JSON `{"status": "...", "error": null}` into `#sse-progress` div during running state. Functionally correct (page reloads on terminal state), but shows unformatted JSON to user. Fix: format SSE data as HTML or use `hx-on::sse-message` to parse JSON and render status.

- **Accept-hero-candidate partial gap**: HTMX swap target `#lp-stage-controls` shows approval counts and buttons, but not the updated hero image. User must reload to see new hero. Fix: target a larger swap area or return full hero card.

- **Dead import in ui/router.py**: `LandingPageRequest` imported from `app.services.landing_page` on line 542 is shadowed by `LPRequest` alias from `app.schemas` on line 543. The line-542 name is never used. Fix: remove unused import.
